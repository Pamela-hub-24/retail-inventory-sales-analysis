# Retail Inventory and Sales Analysis

## Overview
This project analyzes retail sales performance, inventory movement, and customer behavior using Power BI.

## Tools Used
- Power BI
- Power Query
- Data Modeling
- KPI Design

## Key KPIs
- Total Sales Revenue: $323K
- Total Quantity Sold: 5,510 units
- Average Order Value: $322.97
- Gross Income: $15.4K
- Number of Transactions: 1,000

## Key Insights
- Giza branch generated the highest revenue
- E-wallet is the most used payment method
- Food & Beverages contributed the highest gross income
- Members contributed more revenue than normal customers
